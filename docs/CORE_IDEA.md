# The core idea

Let's get real, getfursu.it will never be there, however these are the ideals that should be pursued:

1. Helpful and time saving.
2. Transparent, fair, and safe.
3. Death-proof.

# 1. Helpful and time saving

- Easy to use.
- Easy to find what you're looking for.
- Time saving.
- Up-to-date.
- Rich in data.

# 2. Transparent, fair, and safe

- No paid boosts ("promoted").
- No unverified ratings.
- Safe for minors.
- Difficult for scammers.

# 3. Death-proof

- Open source&data to allow anyone to pick it up if abandoned.
- Easy and cheap to operate.

# And also

- Free, both for the users and for the makers. Because otherwise it had both fewer entries and users. It wouldn't be as helpful as it can be, and no way would it be fair.
